# Node CLI npmrc package

![npm](https://img.shields.io/npm/v/@node-cli/npmrc?label=version&logo=npm)

> Npmrc allows you to toggle different npmrc configuration files on the fly.

## Installation

```sh
> npm install --global @node-cli/npmrc
```

## Examples

Get help

```sh
> npmrc --help
```
